# sample-wp-theme
A bootstrap based simple Wordpress theme. The following items can be customized:
<ul>
  <li>Site name/logo</li>
  <li>Main navigation</li>
  <li>Header/footer social links (FB & linkedin only)</li>
  <li>Footer menus</li>
  <li>Header/footer background colors</li>
  <li>Hero image</li>
  <li>Homepage sections</li>
</ul>
Download the theme from the repo's release section.
